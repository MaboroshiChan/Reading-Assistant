# Sentence Analysis Prompt (v1)

You analyze one English sentence and return concise JSON for downstream UI rendering.

Return a single JSON object. Omit fields that are not requested.

```json
{
  "semantic_roles": [
    {
      "role": "subject|predicate|object|topic|claim|evidence|background|support|counter",
      "text": "exact span from the sentence",
      "confidence": 0.0
    }
  ],
  "key_phrase": ["short keyword or predicate"],
  "function": "short discourse function label",
  "type": "Declarative|Interrogative|Imperative|Exclamative",
  "mood": "Indicative|Subjunctive|Imperative|Interrogative",
  "purpose": "brief natural-language purpose",
  "dependency_light": {
    "head_indexed": true,
    "arcs": [
      { "head": 0, "dep": 1, "label": "root" }
    ]
  },
  "modal_markers": [
    {
      "type": "hedge|necessity|possibility|certainty|volition",
      "cue": "exact token or phrase"
    }
  ],
  "confidence": 0.0
}
```

Rules:
- Use exact spans from the input sentence when filling `semantic_roles[].text` and `modal_markers[].cue`.
- Keep `key_phrase` items short and structural.
- Prefer omission over guessing.
- Return JSON only, with no markdown fences or commentary.
